# 🏋️ Summer Arc — Personal Trainer Digital

> *"Construa seu melhor verão — um treino por vez"*

Summer Arc é uma plataforma web de consultoria fitness que gera treinos e dietas personalizados de acordo com o perfil do usuário. Desenvolvida em equipe como projeto acadêmico/prático, a aplicação funciona 100% no navegador, sem necessidade de backend.

---

## 🚀 Funcionalidades

- **Treino personalizado** gerado automaticamente com base no perfil do usuário
- **Dieta individualizada** com plano alimentar e hidratação recomendada
- **Personalização por gênero** — planos específicos para homens e mulheres
- **Níveis de experiência** — Iniciante, Intermediário e Avançado
- **Técnicas avançadas de treino** — Cluster, Feeder, Back-Off e Aquecimento Específico
- **Divisões de treino** adaptadas à disponibilidade semanal (3, 4 ou 5 dias)
- **Modal de vídeo** para demonstração dos exercícios
- **Frases motivacionais** dinâmicas na tela inicial
- **Design responsivo** — funciona em mobile e desktop

## ▶️ Como rodar

Por ser um projeto front-end puro, basta abrir o arquivo no navegador:

```bash
# Clone o repositório
git clone https://github.com/FlavioEugenio/summer-arc.git

# Abra o arquivo no navegador
Abra o arquivo index.html diretamente no seu navegador
```

> Não é necessário instalar nada ou configurar servidor.
